# Figure 2.2 caption (OSU thesis format, placed below the figure)

Figure 2.2. Overview of the study workflow, from reference data sources through feature construction,
classification, and evaluation. Five embedding configurations and one full spectral configuration
were classified with an identical Random Forest and evaluated against interpreter-labeled reference
cells. Validation sampling and interpretation are developed in Chapter 3.

## Alt text (OSU requires alt text on figures)

A four-band flow diagram in which reference inputs feed a balanced training set, imagery feeds five
embedding configurations and one spectral configuration, a fixed Random Forest produces ten-class
maps, and three evaluation outputs assess accuracy, spatial coherence, and the reliability ceiling,
with the validation reference cells routed to evaluation rather than training.
